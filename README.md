# Wusul - وصول
فكرة المشروع:
تطبيق لتحسين خدمات النقل العام

وصف المشروع:
يهدف هذا المشروع إلى تصميم تطبيق محلي يوفّر معلومات محدثة 
عن مواعيد وسائل النقل العام، مع تنبيهات فورية حول الازدحام أو الأعطال.
يتيح التطبيق للمستخدم معرفة كيفية الوصول إلى أي مكان من خلال عرض جميع الطرق المتاحة،
وأسعار المواصلات، وخط السير، والوقت المتوقع لكل خيار. ويسعى إلى تحسين تجربة النقل العام في المدن المصرية وجعلها أكثر كفاءة وموثوقية.
